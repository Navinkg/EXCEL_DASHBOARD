# 🍔 Swiggy Sales Dashboard using Microsoft Excel

## 📖 Overview

This project presents an interactive **Swiggy Sales Dashboard** built in **Microsoft Excel** to analyze food delivery data. The dashboard provides insights into sales performance, customer behavior, restaurant ratings, delivery trends, and order distribution through dynamic charts, Pivot Tables, and slicers. It enables users to explore the dataset efficiently and make data-driven decisions.

---

## 🎯 Objectives

- Analyze Swiggy sales and order data
- Monitor key business performance metrics
- Identify customer ordering patterns
- Compare restaurant performance
- Visualize trends using interactive charts
- Build an easy-to-use dashboard for business insights
